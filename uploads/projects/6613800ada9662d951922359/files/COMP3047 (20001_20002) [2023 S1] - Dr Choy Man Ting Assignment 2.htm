Assignment 2

Event Management System

Develop an event management system for an NGO using Bootstrap, Express.js, Azure CosmosDB, AJAX, AND VUE.js.

Key Features

The system should support:

  • Adding new events
  • Removing existing events
  • Updating event details
  • Volunteer registration
  • Admin panel
  • First page will show 3 LATEST events.

Technical Requirements

  • Use Bootstrap for responsive layout and styling
  • Build with Express.js and use EJS as the template engine
  • Store data in Azure CosmosDB database
  • Adopt a two-tier architecture, with frontend being built using Vue.js Composition API
  • Implement frontend logic using Vue methods instead of regular JavaScript functions

Evaluation

  • System will be tested on Google Chrome or Safari
  • Use designated GitHub repo (https://classroom.github.com/a/n_7G4CfQ) for Express Backend.
  • Use designated GitHub repo (https://classroom.github.com/a/KLW9KhD0) for Vue SPA Frontend.

Deadline

  • Submit by Nov 17, 10am
  • Email submission not allowed
  • No marks for late submissions
  • Zip files uploaded to GitHub will not be accepted.

Development Guidelines

  • Validate user input on front and back end
  • Handle errors gracefully and provide user feedback
  • Follow coding best practices and style guides
